## Описание

**Movie Library** — это десктопное GUI-приложение на Python (tkinter) для хранения и управления личной коллекцией фильмов.

### Возможности

- ➕ Добавление фильмов с полями: **название**, **жанр**, **год выпуска**, **рейтинг**
- 📋 Отображение коллекции в таблице с чередованием строк
- 🔍 Фильтрация по жанру и диапазону лет
- 🔃 Сортировка по любому столбцу (клик по заголовку)
- 🗑 Удаление выбранного фильма
- ✅ Валидация ввода:
  - Год — целое число в диапазоне 1888 — текущий год + 5
  - Рейтинг — число от 0.0 до 10.0
- 💾 Автосохранение в файл `movies.json`
- 📂 Автозагрузка при запуске

## Примеры использования

### Добавить фильм

1. Введите название в поле **Название** (например: `Начало`)
2. Выберите жанр из выпадающего списка (например: `Триллер`)
3. Укажите год (например: `2010`)
4. Укажите рейтинг (например: `8.8`)
5. Нажмите кнопку **+ Добавить фильм**

### Фильтрация

- Выберите жанр `Фантастика` — таблица покажет только фантастику
- Укажите год **от** `2000` **до** `2015` — только фильмы за этот период
- Нажмите **✕ Сбросить** для отмены всех фильтров

### Сортировка

Кликните по заголовку столбца (`Год`, `Рейтинг`, `Жанр`, `Название`) — список отсортируется по возрастанию.

## Возможные тесты

| Сценарий | Ожидаемый результат |
|---|---|
| Год = `abc` | Ошибка: «Год должен быть целым числом» |
| Год = `1800` | Ошибка: год вне допустимого диапазона |
| Рейтинг = `11` | Ошибка: «Рейтинг должен быть от 0 до 10» |
| Рейтинг = `-1` | Ошибка: «Рейтинг должен быть от 0 до 10» |
| Пустое название | Ошибка: «Введите название фильма» |
| Корректные данные | Фильм добавляется в таблицу и сохраняется в JSON |
| Перезапуск приложения | Все ранее добавленные фильмы загружаются из `movies.json` |
| Фильтр по жанру «Фантастика» | Только фантастика отображается в таблице |
